在 FL Studio 中有多种因素共同决定音符的音量或音高. 要使某种因素成为决定音量或音高的唯一因素, 请将除此之外的因素设为默认或最大值.

简而言之, 此处表格遵循的是 FamiTracker 的思维, 例如一个音符的最终音量为 50%, 决定该结果的只能是一种因素.

算法示例 1: 最终音量 50% = 音符力度 (50%) * 插件主音量 (100%) * 通道音量 (100%) * 混音器输出音量 (100%) * 主控音量 (工程全局, 100%)

算法示例 2: 最终音高 C5 = 音符音高 (C5) * 音符微调音高 (50%, 0 音分, 居中) * 插件主音高 (50%, 0 音分, 居中) * 通道音高 (50%, 0 音分, 居中) * 主控音高 (工程全局, 50%, 0 音分, 居中)
音量 音符力度 音量包络 / LFO (矫音器) 插件主音量 通道音量 混音器输出音量 (效果器) 混音器输出音量 (Master) 主控音量 (工程全局)
音高 音符音高及音符微调音高 音高包络 / LFO (矫音器) 插件主音高 通道音高 ---- ---- 主控音高 (工程全局)
