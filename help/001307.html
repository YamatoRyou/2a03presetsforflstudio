词汇表
单词 说明
OP (Operator) 运算器, Sytrus 中的基本发生器单位, 可产生波形信号或经过 FM (Frequency Modulation, 频率调制) 或 RM (Ring Modulation, 环形调制) 后的波形信号.
WS (Wave Shaper) 波形整形, Sytrus 中的 WS 模块可以将原始波形进行重塑. 与其相似的插件为 Fruity WaveShaper.
Click / Declick 一个波形的起始触发电平不等于 0db 时会产生不同程度的咔哒声, 它可能会影响瞬时动态范围. 音频流中出现一段恒定的波形偏上或偏下则称为直流偏移 (DCO: Direct Current Offset). Sytrus 的每个 OP 都具备 Declick 开关以尽可能避免此问题.
Center Sytrus 的每个 OP 都具备 Center 开关以去除直流偏移使波形居中. 请注意: 它并不能使波形的起始处居中, 因此触发时可能仍然会出现咔哒声.
FILT (Filter) 滤波器, Sytrus 具备 3 个滤波器对上游信号进行过滤.
ENV (Envelope) 包络, 使用线段 (直线或曲线) 对波形的指定目标随时间流逝进行增加或减小, 可以作用于音量 / 音高 / 声像等.
ADSR (Attack / Decay / Sustain / Release) 分别包括 "起始" / "衰减" / "持续" / "释放", 它们是一个完整包络的组成部分.
MML (Music macro Language) 音乐宏语言, 使用特定格式的字符描述音符长度 / 音量 / 音高, 还可以用一串不连续的数字表示包络或波形.
