插件 重要参数 注释
Sytrus DECLICK

■ 移除音符起始嘀嗒声 (起始软化).
B.LIMIT

■ 波段限制波形, 用于获得更干净的声音.
OVERSAMPLING

■ 超采样系数, 用于提升波形成型质量, 有效降低 WS 模块产生的噪音, 但会增加 CPU 占用. 通常不超过 16x.
ENV / LFO

■ 音量包络: ATTACK 与 DECAY 处于同一时间可进一步移除音符起始嘀嗒声.

■ 长度一致原则: 若其中一个包络 / LFO 已启用, 且它的长度超出其它所有包络 / LFO 的每个原始长度, 则所有 OP 的包络 / LFO 长度与该包络的长度相同. 不再需要该包络 / LFO 或发现音符长度异常应及时禁用.

■ 谐波丢失与 B.LIMIT 失效: 当使用通道音高事件; 滑行属性改变音高时, 大幅度滑行将丢失部分谐波; 反之 B.LIMIT 将失效 (除非停用 B.LIMIT 并启用 OVERSAMPLING).

■ 以 MML 规则启用音量包络时, 建议酌情减小插件主控音量.
MOD X / Y

■ 主控面板上的 MOD X / Y 参数值会与音符中的 MOD X / Y 属性值相乘 (例如一个音符的 MOD Y 属性值为 -50%, MOD Y 参数值为 100%, 则最终效果等同于 MOD Y 属性值为 0%, MOD Y 参数值为 50%). 该通道中若只有少数音符应用这些效果, 则应将主控面板上的 MOD X / Y 归零.
Patcher

DirectWave 最大复音数 (通道设置)

■ 在该插件中, 限制最大复音数的情况下快速连奏可能会导致部分音符不响应, 请设置通道最大复音数为允许值的 2 倍. 原因可能是 Native 版本的 DirectWave 存在 Bug.
Dance 跳舞动作

■ 该参数在事件编辑器中会随动作表中定义的动作数量产生不平滑的阶梯, 利用此特性可以模拟 MML.
